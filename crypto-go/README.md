# Crypto Go

Go REST API сервис для криптографических операций с документами: подписание, верификация, шифрование, дешифрование и хеширование.

## Функциональность

| Метод  | Endpoint                 | Описание                                             |
|--------|--------------------------|------------------------------------------------------|
| POST   | `/v2/crypto/sign`        | Подписание документа (PKCS#7, `detached=true/false`) |
| POST   | `/v2/crypto/verify`      | Верификация подписи                                  |
| POST   | `/v2/crypto/encrypt`     | Шифрование документа                                 |
| POST   | `/v2/crypto/decrypt`     | Дешифрование документа                               |
| POST   | `/v2/crypto/hash`        | Вычисление SHA-256 хеша                              |
| GET    | `/v2/crypto/operations`  | Лог операций                                         |

Все POST-запросы принимают файл через `multipart/form-data` (поле `file`).

## Технологический стек

- Go 1.26+
- Standard Library (`net/http`)
- H2 (через h2go)
- Bouncy Castle аналог (`go.mozilla.org/pkcs7`, `go-pkcs12`)
- MIME-детекция (`mimetype`)

## Пререквизиты

1. **Go 1.26+**
2. **Сертификаты** — сгенерируйте PKCS#12 keystore (через `generate_certs.sh` или используйте свой)

## Конфигурация (.env)

| Переменная          | Описание                | По умолчанию            |
|---------------------|-------------------------|-------------------------|
| `SERVER_PORT`       | Порт HTTP               | `8888`                  |
| `KEYSTORE_PATH`     | Путь к PKCS#12 keystore | `../certs/keystore.p12` |
| `KEYSTORE_PASSWORD` | Пароль keystore         | `changeit`              |
